Here is a **flowchart-style diagram** for **Class and Object in Java**.

---

## Flowchart Style Diagram

```text
[Start]
   |
   v
[Class is created]
   |
   |   (Blueprint / Template)
   v
[Object is created using new keyword]
   |
   v
[Object stores data]
   |
   v
[Object uses methods of the class]
   |
   v
[Program output]
```

---

## Another Easy Flowchart

```text
Class Student
   |
   |-- data: name, age
   |
   |-- methods: display()
   |
   v
Create object:
Student s1 = new Student();
   |
   v
Set values:
s1.name = "Amit"
s1.age = 21
   |
   v
Call method:
s1.display()
```

---
